# SKAICONNECT Website - Quick Start Guide

## 🚀 Getting Started

Simply open `index.html` in your web browser to view the website!

### Option 1: Double-Click
- Navigate to the `skaiconnect` folder
- Double-click `index.html`
- The website will open in your default browser

### Option 2: Right-Click
- Right-click on `index.html`
- Select "Open with" → Choose your preferred browser

### Option 3: Drag & Drop
- Drag `index.html` into an open browser window

## 🔧 Customization

### To Change Colors
Edit `css/styles.css` and modify the `:root` variables:
```css
:root {
  --primary-red: #B91C1C;
  --accent-gold: #D97706;
  /* etc. */
}
```

### To Change Content
Simply edit the HTML files with any text editor:
- Notepad (Windows)
- TextEdit (Mac)
- VS Code (Recommended)
- Sublime Text
- Any HTML editor

### To Add Your Own Images
1. Place images in the `images/` folder
2. Update the `<img src="...">` tags in HTML files
3. Example: `<img src="images/your-photo.jpg">`

## 🌐 Deployment

### Option 1: GitHub Pages (Free)
1. Create a GitHub repository
2. Upload all files
3. Enable GitHub Pages in settings
4. Your site will be live at `username.github.io/repo-name`

### Option 2: Netlify (Free)
1. Drag the `skaiconnect` folder to netlify.com/drop
2. Your site goes live instantly
3. Get a free subdomain or use your own

### Option 3: Traditional Web Hosting
1. Upload all files via FTP
2. Ensure `index.html` is in the root directory
3. Access via your domain
