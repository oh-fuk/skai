  🚀 QUICK GUIDE: Add New Blog in 3 Steps
═══════════════════════════════════════════════════════════════

STEP 1: Add Blog Card to blog.html
───────────────────────────────────────────────────────────────
1. Open: skaiconnect/blog.html
2. Find: <div id="blogGrid">  (around line 74)
3. Scroll to BEFORE the closing </div>
4. Copy template from BLOG-TEMPLATE.html
5. Paste it there
6. Replace:
   - YOUR_CATEGORY → e.g., "Cloud Migration"
   - YOUR_BLOG_TITLE → e.g., "10 Tips for Cloud Success"
   - YOUR_IMAGE_URL → e.g., "https://images.unsplash.com/photo-xxx"
   - YOUR_DATE → e.g., "May 1, 2024"
   - YOUR_SHORT_DESCRIPTION → 2-3 sentences
   - blog-post-NUMBER.html → e.g., "blog-post-10.html"
7. Save file

───────────────────────────────────────────────────────────────

STEP 2: Create Full Blog Post Page
───────────────────────────────────────────────────────────────
1. Copy: blog-post-1.html
2. Rename to: blog-post-10.html (or next number)
3. Open the new file
4. Change:
   Line 6:   <title>Your Title | SKAICONNECT Blog</title>
   Line 50:  Category tag
   Line 51:  Date
   Line 54:  <h1>Your Blog Title</h1>
   Line 56:  <img src="your-image-url">
   Line 58+: Replace all content with your article
5. Save file

───────────────────────────────────────────────────────────────

STEP 3: Test It!
───────────────────────────────────────────────────────────────
1. Open blog.html in browser
2. See your new blog card ✓
3. Click category filter → Your blog appears/hides ✓
4. Search for keywords → Your blog is searchable ✓
5. Click "Read More" → Opens your full article ✓

⚠️ COMMON MISTAKES:
───────────────────────────────────────────────────────────────
❌ Category name doesn't match: "Cloud Migration" vs "cloud migration"
   ✅ Must match EXACTLY (case-sensitive)

❌ Added blog outside #blogGrid
   ✅ Must be INSIDE <div id="blogGrid">

❌ Link doesn't match filename: href="blog-10.html" but file is "blog-post-10.html"
   ✅ Names must match exactly

❌ Forgot data-category or data-title
   ✅ Both are required for search/filter to work
